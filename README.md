# Divorce Prediction
This repository contains Machine Learning models for Divorce Prediction using the [Divorce Predictions Data Set](https://archive.ics.uci.edu/ml/datasets/Divorce+Predictors+data+set) from UCIS Machine Learning Repository. 
This dataset is based on [Dr. John Gottman's](https://en.wikipedia.org/wiki/John_Gottman)'s questionnaire that he created throughout his career as Couple Therapist.

The purpose of this project is to apply Supervised Learning techniques to this data as a way of studying Machine Learning algorithms.
